## 总体格式
格式："archive_diff":"|1=249584>>>249711|89-12572=nil>>>{}"
非表格存档示例：|槽位=原值>>>变化后的值
表格存档示例：|槽位=原值>>>变化后的值


## 表格存档变动示例
### Y3编辑器表格存档规则
1. 必须一个个键值设置，不能直接设置某个key对应的value为有内容的表(空表可以)
2. 多层嵌套表格结构，必须先设置空表，才能设置该空表内的键值，不能直接设置多层key的键值
3. 表格存档最后3层嵌套，每层嵌套最多500个kv

### 日志：|89-12572=nil>>>{}
实际变化前：
{
    "89":{}
}

实际变化后：
{
    "89":{
        "12572":{}
    }
}

### 日志：|89-12572-宝石ID=nil>>>20014
实际变化前：
{
    "89":{
        "12572":{}
    }
}

实际变化后：
{
    "89":{
        "12572":{
            "宝石id":20014
        }
    }
}


### 日志：89-12514={'\\xe6\\xa0\\x8f\\xe4\\xbd\\x8d1': 0}>>>nil
**设置某个键对应的值为nil，相当于删除该键值对**
实际变化前：
{
    "89":{
        "12514":{'\\xe6\\xa0\\x8f\\xe4\\xbd\\x8d1': 0}
    }
}

实际变化后：
{
    "89":{
        "12514":{}
    }
}
